start 和 run 的区别

  • Thread.start() 启动新的线程，线程处于就绪（可运行）状态，线程获取到 CPU 资源之后，就执行该线程相应的 run 方法。同一个线程的 start 方法不能被重复调用。
  • Thread.run() 方法内容是线程体，线程具体的执行步骤。run 只是一个方法调用，可以被重复多次调用，不会开辟新的线程。
